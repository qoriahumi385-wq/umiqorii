✦ Proyek

Proyek yang dikerjakan selama studi di UNIKMA Cilacap.

🏆 Sistem Informasi Jadwal Olahraga Sekolah

Aplikasi web pengelola jadwal latihan, hasil pertandingan, dan data atlet. Dibangun dengan PHP, MySQL, dan Bootstrap.

📊 Dashboard Monitoring Prestasi Siswa

Menampilkan data prestasi dalam grafik interaktif (Chart.js + PHP) untuk membantu guru memantau perkembangan siswa secara real-time.

🌐 Website Profil Sekolah Dinamis

Website sekolah dengan CMS sederhana. Menampilkan info sekolah, berita, dan galeri kegiatan secara dinamis.

📱 Aplikasi Pencatat Latihan Harian

Aplikasi mobile-first untuk mencatat aktivitas olahraga harian menggunakan HTML, CSS, JavaScript, dan Firebase.

🎨 Prototype UI/UX Portal Kampus

Desain antarmuka portal informasi mahasiswa di Figma dengan UX Research dan high-fidelity prototype interaktif.
